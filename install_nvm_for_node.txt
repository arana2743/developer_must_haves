installing nvm which is node version manager for ubuntu or linux system.

link for github link
https://github.com/creationix/nvm

wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.33.6/install.sh | bash

source ~/.bashrc

command -v nvm  ==> nvm if properly installed.

usage:

1. listing versions

>nvm ls which versions are installed
>nvm ls-remote for available node versions 
>nvm install node[version]
